AI agents run untrusted code and are steerable by an attacker through the prompt. Model guardrails aren't a security boundary, and a container shares the host kernel. This series builds the boring, important thing that makes agents safe to run — **infrastructure that assumes the agent is hostile** — and every part ends with a **"🐛 What bit me"** box so the gotchas that cost me hours don't cost you any.
